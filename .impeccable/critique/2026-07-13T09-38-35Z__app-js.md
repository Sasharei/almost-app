# Overall Impression

# Priority Issues

## P1 — Accessibility architecture

**Why it matters:** интерфейс может быть физически непригоден для пользователей Dynamic Type, Reduce Motion и screen reader. Это затрагивает каждую поверхность, а не один экран.

**Fix:** включить font scaling с управляемыми max multipliers; добавить reduced-motion token и platform listener; дать tabs, switches, sliders, icon buttons и gestures полноценные role/label/state/value; поднять hit targets до 44 pt iOS и 48 dp Android; исправить foreground contrast всех PRO accents.

**Suggested command:** $impeccable harden

## P1 — Information architecture and hierarchy

**Why it matters:** Feed конкурирует сам с собой, а Progress и Profile являются инвентарями функций, а не task-oriented destinations. Пользователь видит уровень, streak, rating, cat, carousel, subscriptions, coins, budgets, challenges, maps, themes, locale, history и privacy без ясного порядка.

**Fix:** оставить на Home один эмоциональный момент и одно следующее решение; превратить Progress в Overview с drill-in destinations для Goals и Insights; разделить Profile и Settings; скрыть редкие настройки за grouped rows.

**Suggested command:** $impeccable distill

## P1 — Navigation and modal hierarchy

**Why it matters:** скрытые swipe/long-press/fridge affordances и множество full overlays повышают ошибочность и разрушают контекст. Полностью locked Budget tab ощущается как навигационная ловушка.

**Fix:** одна понятная short-tap функция FAB и видимое меню для дополнительных действий; доступные эквиваленты swipe; native tab semantics; toast/snackbar для пассивного success, sheet для редактирования, alert для destructive, full-screen только для редких milestones; limited-useful Budget preview для free user.

**Suggested command:** $impeccable clarify

## P2 — Motion overload

**Why it matters:** 69 perpetual loops делают всё одинаково важным, расходуют внимание и могут ухудшать perceived performance и самочувствие.

**Fix:** один ambient animation максимум на экран; убрать вечные shimmer/breathe/mist у chrome; оставить spring только для gesture continuity; exit быстрее enter; celebrations заканчиваются и успокаиваются; Reduced Motion заменяет перемещение на crossfade.

**Suggested command:** $impeccable animate

## P2 — Fragmented visual system and contrast

**Why it matters:** тысячи style keys, сотни цветов и десятки radius values создают drift между экраном, modal, card и paywall. Light muted text имеет contrast 3.72:1; white foreground проваливается на emerald 2.36:1, sunset 2.57:1, gold 2.15:1 и aqua 2.57:1.

**Fix:** semantic tokens вместо raw colors; три surface elevations; четыре radius tokens; единая type ramp; readable foreground resolver для всех accents; paywall наследует активную theme/accent систему.

**Suggested command:** $impeccable colorize

# Persona Red Flags

**Jordan, first-timer:** до понимания основного loop должен расшифровать personas, health coins, fridge, tycoon и Progress/Rewards. На Home не очевидно, что сделать первым.

**Casey, distracted one-handed user:** horizontal carousels, long vertical scrolls, hidden swipes, variable FAB action и repeated overlays ухудшают восстановление после прерывания.

**Sam, accessibility-dependent user:** глобальная блокировка Dynamic Type, отсутствие Reduce Motion, missing tab/switch semantics, маленькие controls и недостаточный accent contrast способны сделать ключевые flows недоступными.

# Minor Observations

- Liquid Glass на iOS 26+ — сильная platform-aware инвестиция; Android fallback должен эволюционировать в Material 3 tonal surfaces, а не имитировать iOS glass.
- Pill button высотой 42 ниже рекомендуемого Android target и пограничен для iOS.
- FAB long press открывает два больших действия, но affordance нигде явно не объяснён.
- Некоторые debt/transfer inputs полагаются на placeholder-only labels.
- Paywall palette визуально не связана с текущим PRO accent.
- Light secondary text стоит затемнить; border contrast 1.16:1 сам по себе допустим только когда граница не несёт смысл.
- Печальные/голодные реакции Алми после траты следует переписать из shame в compassionate recovery: факт → поддержка → одно маленькое действие.
